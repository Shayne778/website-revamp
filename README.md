# website-revamp (Static GitHub Pages)

This repo is now a plain static site.

## Required GitHub Pages settings
- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/ (root)**

## Important
If your local branch is not `main`, push this branch to `main` explicitly:

```bash
git push -u origin HEAD:main
```

The homepage is served from `index.html` at repo root.
